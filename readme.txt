お客様のサービスに合わせて使用できるカスタマイズチャットボットです。

== Description ==

outaigateをプラグインすると、すぐにチャットボット使うことが出来ます。お客様のサービスに合わせて使えるカスタマイズ性の高いチャットボットとして簡単にチャットボットを製作し、自社サイトに埋め込めます。
チャットボットがインストールされると、自社サイトに接続した顧客とコミュニケーションするために、ワードプレスとoutaigateサーバーと通信し、管理者がoutaigateに接続して顧客とコミュニケーションすることができます。

== Installation ==

•   管理者ダッシュボードに接続
•  「プラグイン」メニューをクリック
•  「新規追加」をクリック
•  「outaigate」検索
•  「今すぐインストール」をクリック
•  「プラグイン」メニューに移動
•   outaigateにログイン
•  「設定」ｰ「相談ボタン設置」からチャットボットプラグインのためのブランドキーをコピー
•   ブランドキーを入力し完了

== Frequently Asked Questions ==

「よくある質問」

・outaigateは無料ですか？
  →まずは14日間無料体験が出来ます。それ以降は1アカウント当たり月4,900円のスクリプションとしてお使いいただけます。

・チャットボット以外に活用できる機能はありますか？
  →チャットボットは365日24時間運営することができ、有人チャットに切り替えてwebチャットやLINE、メールでの対応も可能です。そして、顧客のニーズをいち早く把握することができます。

・チャットボットのカスタムは可能ですか？
  →はい、チャットボットの色や、イメージなどを変更することが出来ます。チャットボットにイメージ画像やリンクの貼り付けが可能なため幅広くお使いいただけます。
